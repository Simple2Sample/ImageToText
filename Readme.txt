install pynput, opencv2-python, pyperclip,pytesseract, and Pillow
Make sure pytesseract lies in 'C:\Program Files (x86)\Tesseract-OCR\\tesseract.exe' if you are on Windows.
Linux users should remove this line as the library will be installed automagically.

https://towardsdatascience.com/read-text-from-image-with-one-line-of-python-code-c22ede074cac


How to use:
1. Start the Python script.
2. Use snippet tool or copy an image that you want the program to read.
3. Press left Alt to start the program.

Warning! Pressing left alt twice or processing anything other than a picture crash the program.
